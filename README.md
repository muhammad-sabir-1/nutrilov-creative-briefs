# Nutrilov — Creative Briefs

Static idea guides for Nutrilov's creative pipeline. Each subfolder is one brand × product × competitor batch, deployed to GitHub Pages as a read-only reference for designers and AI generators.

## Pages

- **[eleat-statics/](eleat-statics/)** — Protein Granola × Eleat. 24 static ad briefs adapted from Eleat's highest-performing FB Ad Library creatives.

## How to read a brief

Each card on a page shows the source competitor ad as the thumbnail. Click → side panel opens with:

- **Format** — dimensions to produce
- **Copy** — image overlay text, social caption, CTA
- **Visual** — what vibe to match, what to swap (deep red Nutrilov pack, granola bowl, etc.)
- **Variants** (when present) — alternate headline / dimension combos to test
